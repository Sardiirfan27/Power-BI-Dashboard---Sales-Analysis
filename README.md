# Sales Performance Dashboard

Proyek ini merupakan proyek untuk pelatihan membuat **dashboard interaktif untuk memantau dan menganalisis performa penjualan hingga proses pengiriman**, dirancang untuk membantu tim bisnis dan manajemen dalam pengambilan keputusan berbasis data. Dashboard ini dibuat dengan beberapa halaman utama, masing-masing menyoroti aspek tertentu dari performa penjualan.

[![Watch the demo](https://img.youtube.com/vi/rVjrxNV_qKc/0.jpg)](https://www.youtube.com/watch?v=rVjrxNV_qKc)


## Halaman Dashboard

1. **Sales Dashboard**
   Menampilkan ringkasan performa penjualan secara keseluruhan, termasuk total penjualan, tren penjualan per periode, total profit berdasarkan produk dan metrik utama lainnya.

2. **Product Analysis 1**
   Analisis mendalam untuk keseluruhan produk yang mencakup menampilkan performa penjualan berdasarkan wilayah, kategori produk, dan sub kategori produk.

3. **Product Analysis 2**
   Analisis mendalam tentang produk yang terjuah yang berfokus pada analisa hubungan atau pengaruh antara diskon dan total penjualan, hubungan antara diskon dan profit, stok, profit margin, dan lainnya.

4. **Shipping Dashboard**
   Memvisualisasikan data pengiriman, termasuk jenis pengiriman, biaya pengiriman, dan total pengiriman di setiap wilayah.

## Fitur Utama

* Dashboard interaktif multi-halaman dengan menerapkan DAX dan Dynamic Filter.
* Visualisasi data yang jelas dan mudah dipahami.
* Mendukung pengambilan keputusan strategis terkait penjualan, produk, dan pengiriman.
